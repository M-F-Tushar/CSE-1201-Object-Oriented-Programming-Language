**Q: How do I install the C++ compiler?**
- See the [GETTING_STARTED.md](GETTING_STARTED.md#installing-c-compiler) guide for detailed instructions for Windows, macOS, and Linux.

**Q: How do I install Java?**
- See the [GETTING_STARTED.md](GETTING_STARTED.md#installing-java) guide for JDK installation instructions.

- Check that you have the correct compiler/JDK version installed

- Compare with working examples in the repository
